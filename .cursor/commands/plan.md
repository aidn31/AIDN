# Create Implementation Plan

Based on the current conversation, create a structured plan document at `docs/plans/FEATURE_NAME_plan.md` with:

## Feature Description
[What are we building?]

## User Story
[Why does the user need this?]

## Technical Approach
[How will we build it?]

## Files to Modify
[List all files that need changes]

## Task Breakdown
- [ ] Task 1
- [ ] Task 2
- [ ] Task 3
(etc.)

## Testing Strategy
[How will we validate this works?]

## Success Criteria
[What defines "done"?]
